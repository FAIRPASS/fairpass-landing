# FAIRPASS Journal — SNS 플랫폼별 콘텐츠 전략 가이드

> 관리자 페이지에서 SNS 초안 자동 생성 후, 이 가이드를 참고해 검토·수정하세요.
> 핵심 원칙: **블로그 링크 클릭을 유도하되, 가장 궁금한 부분은 공개하지 않기**

---

## 공통 원칙

- CTA는 항상 마지막에 한 번만 (링크는 게시물 외부나 bio 링크)
- 이모지는 LinkedIn KO/EN에서 절제, 인스타·블로그는 자유롭게
- 해시태그: 인스타 10~15개, LinkedIn 3~5개, 네이버 블로그 5~10개
- 모든 채널에서 "FAIRPASS" 노출은 자연스럽게, 광고처럼 보이지 않게

---

## 1. LinkedIn (한국어)

**타겟**: B2B 의사결정자, 행사 담당자 시니어급, HR/마케팅 팀장

**성격**: 전문적, 인사이트 중심, 커리어/업무 관련성 강조

**최적 포맷**:
```
[핵심 인사이트 or 충격적 수치 — 첫 줄에서 멈추게]

본문 2~3단락:
- 문제 제기 또는 배경
- 핵심 발견/결과 (수치 포함)
- 한 가지 실천 가능한 팁

[링크 클릭 유도 문구]
👉 전체 내용: [제목] (링크)

#행사운영 #이벤트기술 #FAIRPASS #B2B #컨퍼런스
```

**카테고리별 포커스**:
| 카테고리 | 강조 포인트 |
|---------|------------|
| FAIRPASS 이야기 | 창업/성장 인사이트, 의사결정 배경 |
| 활용법 | 업무 효율화, 시간/비용 절감 수치 |
| 운영 사례 | 레퍼런스, 비포/애프터 지표 |
| 지속가능한 행사 | ESG 전략, 기업 가치 연결 |
| 업계 트렌드 | 업계 동향, 전략적 시사점 |
| FAIRPASS 소식 | 제품 업데이트, 파트너십 의미 |

## 2. LinkedIn (English)

**타겟**: 글로벌 이벤트 담당자, 아시아 APAC 시장, 해외 파트너 후보

**성격**: 더 직접적, 글로벌 컨텍스트 강조, 한국/아시아 레퍼런스 부각

**최적 포맷**:
```
[English hook — concise, data-driven]

2~3 short paragraphs:
- Global/APAC context
- What we learned / what changed
- One actionable insight

Read more → [title] (link)

#EventTech #EventManagement #FAIRPASS #APAC #EventPlanning
```

**주의사항**:
- 한국 기관명은 영문으로 병기 (예: Hyundai Motor Company)
- 수치는 글로벌 비교 맥락으로 (예: "at ADB's annual conference...")
- 직역 금지 — 영어권 독자 관점으로 재작성

## 3. Instagram (한국어)

**타겟**: 행사 담당자 실무자, 이벤트 기획사 직원, 20~40대

**성격**: 비주얼 우선, 짧고 강렬, 공감·저장 유도

**최적 포맷**:
```
[첫 줄 — 스크롤 멈추게 하는 한 문장]
(이모지 적극 활용 ✨)

핵심 내용 3~5줄 (짧게, 줄바꿈 많이)

.
.
.

👇 전체 가이드는 링크에서
(bio 링크 또는 스토리 링크)

#이벤트기획 #행사담당자 #FAIRPASS #컨퍼런스운영
#온라인접수 #QR체크인 #행사기획 #이벤트플랫폼
#행사자동화 #명찰출력 [상황별 추가 태그]
```

**카테고리별 포커스**:
| 카테고리 | 인스타 전략 |
|---------|------------|
| FAIRPASS 이야기 | 감정 공감, 창업자 인간적 면모 |
| 활용법 | 팁 카드뉴스 형식, 저장 유도 |
| 운영 사례 | 비포/애프터 비주얼, 수치 강조 |
| 지속가능한 행사 | 환경 이미지, 수치 인포그래픽 |
| 업계 트렌드 | "몰랐지?" 포맷, 공유 유도 |
| FAIRPASS 소식 | 신기능 하이라이트, 호기심 유발 |

**비주얼 제안** (이미지 없을 때 텍스트로 대체):
- 카드뉴스: 핵심 수치나 팁 1~3개 시각화
- 현장 사진: 행사 현장 + 키오스크/QR 화면
- 그래프/비교: 이전/이후 단순 도표

## 5. 네이버 블로그

**타겟**: 검색 유입 — "행사 접수 방법", "QR 체크인 어떻게" 검색하는 실무자

**성격**: SEO 최우선, 정보량 충분히, 이미지 필수

**최적 포맷**:
```
[제목 — 검색어 포함, 클릭 유도]

요약: [2~3줄로 핵심 정리]

본문:
- 소제목 3~5개 (H2 수준)
- 각 섹션 이미지 1장 이상
- 수치, 표, 목록 적극 활용
- 내부 링크: fairpass.world 링크 2~3개

마무리:
"더 자세한 내용은 FAIRPASS 공식 저널에서 확인하세요"
[블로그 링크]

태그: #FAIRPASS #행사접수 #QR체크인 #이벤트관리 #온라인접수
      #행사담당자 #컨퍼런스운영 #무인명찰 [주제별 추가]
```

**네이버 SEO 팁**:
- 제목에 "[연도] + 키워드 + 방법/가이드" 포함 (예: "2026 행사 온라인 접수 세팅 방법")
- 이미지 alt 텍스트에 키워드
- 본문 2,000자 이상 권장
- 블로그 발행 후 네이버 서치어드바이저에 URL 제출

## 자동 생성 초안 검수 체크리스트

- [ ] 첫 줄이 스크롤/클릭을 멈추게 하는가?
- [ ] 가장 궁금한 내용은 공개하지 않고 링크로 유도하는가?
- [ ] 수치/사례가 구체적인가? (막연한 표현 수정)
- [ ] CTA가 자연스럽게 이어지는가?
- [ ] 해시태그가 타겟 독자와 맞는가?
- [ ] LinkedIn EN은 직역이 아닌 재작성인가?
- [ ] 네이버 블로그는 검색 키워드가 제목에 포함됐는가?
